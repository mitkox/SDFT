# Contributing

Thanks for your interest in contributing!

## Development setup

- Python 3.12 recommended
- Install dependencies: `pip install -r requirements.txt`
- Install PyTorch separately (see `README.md`)

## Code style

- Keep changes focused and easy to review.
- Prefer clear, explicit configuration over hard-coded constants.
- Avoid committing large generated artifacts (datasets, checkpoints, logs).
